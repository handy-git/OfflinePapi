# OfflinePapi

#### 介绍

一款有点好用的离线变量插件

1. 很多变量不支持离线玩家的查询
2. 这个插件很好的改进了这点，在玩家在线的时候记录变量值，在离线后也可以查询出来

#### 示例

例如你在config中配置了变量 %playerTitle_number%

只需要使用 %offlinePapi_玩家名_playerTitle_number% 就可以获取到玩家离线版本的变量
玩家名使用 me 表示查询自己
